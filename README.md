# Nabda — Releases

Official APK distribution for **Nabda (نبضة)** — an Arabic-first, offline health &
athkar companion app for Android.

## ⬇️ Download the latest version

➡️ **[Latest release](https://github.com/iOsa01/nabda-releases/releases/latest)**

Stable direct link (always serves the newest APK):

```
https://github.com/iOsa01/nabda-releases/releases/latest/download/nabda-latest.apk
```

## 📲 Install

1. Download the APK on your Android phone.
2. Open it. If prompted, allow **“Install unknown apps”** for your browser / file manager.
3. Install. To update later, just install the newer APK **over** the existing one —
   your data is preserved (all releases are signed with the same key).

## ✅ Verify integrity (optional)

Every release lists a **SHA-256** checksum. Verify the downloaded file:

```
sha256sum nabda-latest.apk
```

## 🔄 In-app updates

The app periodically reads `latest.json` in this repository and notifies you when a
newer `versionCode` is available, linking you back here. No account, no tracking.

Source code is maintained in a private repository. This repo hosts release binaries only.
